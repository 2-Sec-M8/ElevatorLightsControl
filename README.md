# ElevatorLightsControl

Controls a Govee LED strip for elevator lighting behaviors.

## Configure

Set environment variables before running:

- `GOVEE_API_KEY`
- `GOVEE_MAC_ADDRESS` (optional, defaults to `D0:6A:D4:AD:FC:FF:A5:81`)
- `GOVEE_MODEL` (optional, defaults to `H618A`)

## Run

```bash
python main.py
```

This powers the strip on and applies the **Elevator Up** effect scene.
